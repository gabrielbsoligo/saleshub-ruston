⚡ Importador MKTLAB → SalesHub

Importe leads do CRM V4 (mktlab.app) para o SalesHub com 1 clique.

Como instalar

1 Mostre a barra de favoritos

Aperte Ctrl+Shift+B no Chrome para mostrar a barra de favoritos.

2 Arraste o botão abaixo para a barra de favoritos:

⚡ Importar → SalesHub

Arraste este botão vermelho para a barra de favoritos do Chrome.

3 Use no MKTLAB

Abra um lead no mktlab.app → Clique no bookmarklet na barra de favoritos → Lead é criado automaticamente no SalesHub + Kommo.

⚠️ Importante: Você precisa estar logado no SalesHub para o import funcionar. O lead é criado automaticamente e a página redireciona para a lista de leads.

📦 Importador em Lote

Importe uma coluna inteira do MKTLAB para o SalesHub. Detecta duplicatas automaticamente.

1 Arraste o botão abaixo para a barra de favoritos:

📦 Importar Lote → SalesHub

Arraste este botão azul para a barra de favoritos.

2 Use na pagina de Leads do MKTLAB

Abra mktlab.app/crm/leads (Kanban view) → Clique no bookmarklet → Painel lateral abre → Login no SalesHub → Selecione coluna → Preview → Importe!

⚠️ Importante: Use na pagina de Leads (view Kanban). O script detecta as colunas do funil automaticamente. Leads duplicados (mesmo mktlab_id, link ou empresa) sao identificados e pulados.

O que é extraído automaticamente

  • Nome da empresa
  • Nome do contato
  • Celular
  • Email
  • CNPJ
  • Canal de aquisição (BlackBox, LeadBroker, etc)
  • Canal de origem (Google, Facebook, etc)
  • Faturamento
  • Produto
  • Valor do lead
